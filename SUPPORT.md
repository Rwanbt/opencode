# Support

## This is a fork

This repository is a **personal fork** of [anomalyco/opencode](https://github.com/anomalyco/opencode).

For issues with the **upstream project** (core agent, providers, TUI, SDK), please open issues in the upstream repository:  
👉 https://github.com/anomalyco/opencode/issues

---

## Fork-specific issues

For issues specific to **fork additions** (mobile on-device inference, Hexagon NPU, Adreno OpenCL, desktop TTS/STT, on-device toolchain):

1. Check [KNOWN_ISSUES.md](KNOWN_ISSUES.md) first — common problems are documented there.
2. Open an issue in this repository with the label `fork-specific`.
