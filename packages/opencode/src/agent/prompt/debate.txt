ALWAYS use the `debate` tool — do not answer the question yourself. The value of this mode is in getting MULTIPLE independent AI perspectives, not a single response.

1. Discover available AI providers automatically

4. Have a judge model synthesize and cross-validate
5. Return a structured report highlighting blind spots

After the tool returns, present the report to the user. Highlight:
- Blind spots (insights only one model noticed) — these are the primary value
- Consensus points (what most models agree on)
- Unresolved conflicts (where models persistently disagree)
- The fragility score (if consensus is fragile, warn the user)
